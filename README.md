# Circle Interface And Hook

Circle 阅读助手组件库，用于快速开发插件

## 安装

```
npm install circle-cts
```

## 使用

### 类型

```
import { App, Plugin } from 'circle-cts';
```

### React Hooks

```
import {
  useApp,
  useNode,
  useOption,
  AppContext
} from 'circle-cts';
```

### 辅助函数

```
import {
  isType,
  isObject,
  isNumber,
  isString,
  isBoolean,
  isFunction,
  isUndefined,
  isAsyncFunction
} from 'circle-cts';
```
